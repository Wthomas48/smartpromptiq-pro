## Code Stacking Strategy

Allow users to stack codes for more resources:

| Codes | Tier 1 | Tier 2 | Tier 3 |

| 1 code | 500 tokens | 2,000 tokens | 5,000 tokens |
| 2 codes | 1,000 tokens | 4,000 tokens | 10,000 tokens |
| 3 codes | 1,500 tokens | 6,000 tokens | 15,000 tokens |
| 5 codes | - | - | 25,000 tokens |

**Bonus for stackers:**
- 2+ codes: +20% bonus tokens
- 3+ codes: +30% bonus tokens + priority support
- 5 codes: +50% bonus tokens + white-label + dedicated account manager

## Pricing Psychology

### Why These Prices Work:

1. **$49 Tier 1** - Impulse buy threshold, low risk
2. **$99 Tier 2** - Sweet spot, most value perception
3. **$199 Tier 3** - Serious users, agencies

### AppSumo Audience Expectations:
- Average LTD purchase: $59-$99
- Stacking buyers prefer 3-code deals
- Agencies look for $150-$250 range

## What to EXCLUDE from LTD (Future Upsells)

Keep these for paid upgrades later:

1. **Enterprise API** - Full API access, webhooks
2. **Unlimited Teams** - More than 5 seats
3. **Custom AI Training** - Fine-tuned models
4. **Dedicated Infrastructure** - Own processing queue
5. **SLA Guarantee** - 99.9% uptime commitment
6. **Custom Integrations** - Zapier, Slack, etc.
7. **Advanced Analytics** - Detailed usage reports
8. **Priority Processing** - Skip queue
9. **Concierge Onboarding** - 1-on-1 setup
10. **Token Top-ups** - One-time token purchases

## Lifetime Deal Terms

### What "Lifetime" Means:
- Access for as long as the product exists
- Monthly token allocation renews forever
- Feature updates included
- Major new modules may be separate

### Fair Usage Policy:
- No reselling accounts
- No automated/bot usage
- Personal or business use only (not resale)
- Tokens don't roll over month-to-month

### Refund Policy:
- 60-day money-back guarantee (AppSumo standard)
- No questions asked
